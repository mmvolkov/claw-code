# AGENTS.md

Этот файл содержит рекомендации для Codex (`Codex.ai/code`) при работе с кодом в этом репозитории.

## Обнаруженный стек
- Языки: Rust.
- Фреймворки: поддерживаемые starter-маркеры не обнаружены.

## Проверка
- Запускайте проверку Rust из `rust/`: `cargo fmt`, `cargo clippy --workspace --all-targets -- -D warnings`, `cargo test --workspace`
- В репозитории присутствуют и `src/`, и `tests/`; при изменении поведения обновляйте обе поверхности согласованно.

## Структура репозитория
- `rust/` содержит Rust workspace и активную реализацию CLI/runtime.
- `src/` содержит исходные файлы, которые должны оставаться согласованными с сгенерированными инструкциями и тестами.
- `tests/` содержит проверочные поверхности, которые нужно просматривать вместе с кодовыми изменениями.

## Рабочие договоренности
- Предпочитайте небольшие, удобные для ревью изменения и держите сгенерированные bootstrap-файлы согласованными с реальными workflow репозитория.
- Общие значения по умолчанию храните в `.Codex.json`; `.Codex/settings.local.json` используйте только для локальных переопределений на конкретной машине.
- Не перезаписывайте содержимое `AGENTS.md` автоматически; меняйте его осознанно, когда меняются workflow репозитория.
